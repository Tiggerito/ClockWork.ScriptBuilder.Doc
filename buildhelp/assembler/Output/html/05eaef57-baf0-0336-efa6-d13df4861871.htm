TrySetLayout(ScriptLayout, Boolean)
Attempt to set the layout. General policy is to only set the layout if it has not been set yet. The presedence of layout control is: Constructor Based Layout if supplied directly by user (e.g. specifically created in script structure with layout) Constructor Based Layout if supplied by parent item (e.g. created by parent item with a layout) Set by Parent after contruction (e.g. parent has a preferred layout) The items Default layout Initially the layout is set to Auto which indicates Default: use this if you want to
